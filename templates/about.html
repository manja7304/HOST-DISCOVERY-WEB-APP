About This Project

The Host Discovery App is a Flask-based web application that scans a given subnet and identifies active hosts on the network.

Developed by Darshan as a Final Year BCA Project, it uses Python, Bootstrap, and Flask. It supports real-time network pinging and a visually attractive frontend.

  • Framework: Flask
  • Frontend: Bootstrap 5, FontAwesome, Google Fonts
  • Scanning: Python IP + subprocess ping
